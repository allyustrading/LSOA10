# Liuli Society of America — Premium Eco Nonprofit Website

A GitHub-ready React + Vite website for **Liuli Society of America**.

## Theme direction
- premium eco nonprofit design
- full-screen hero and wide landscape banners
- nature protection and environmental stewardship focus
- refined institutional tone
- nonprofit-safe content language

## Included pages
- Home
- About
- Our Approach
- Nature & Wellness Pathways
- Research & Innovation
- Community
- Support Us
- Contact

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Hosting
This project uses `base: './'` in Vite config, making it easier to deploy as a static website.
